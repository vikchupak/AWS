# CloudWatch

- CloudWatch Logs is a service which can accept logging data, store it and monitor it.
- Can be used for:
  - ClodWatch logs
  - ClodWatch metrics
  - CloudWatch alarms
  - CloudWatch events
- CloudWatch Stream - Logs from the same source
